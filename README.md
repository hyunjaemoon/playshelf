# playshelf

Online social cataloging service for games. A full-stack application for discovering, searching, and cataloging video games using the IGDB (Internet Game Database) API.

## Tech Stack

**Backend**: Rust with Axum web framework, Tokio async runtime, and IGDB API integration via Twitch OAuth2 authentication.

**Frontend**: Dioxus cross-platform UI framework.
